## 📌 About The Project

**Ezy-Parking** is a comprehensive smart parking management system designed to alleviate the growing problem of parking space scarcity in metropolitan cities. It provides intelligent solutions for the short-time rental of empty spaces (commercial parking lots, apartments, office spaces, etc.) to help users find safe, affordable, and accessible parking while on the go.

By eliminating the need for aimless driving to find a parking spot, Ezy-Parking reduces traffic congestion, prevents unauthorized roadside parking, and ensures a smoother daily commute. In our pilot testing, this platform **reduced average parking search time by ~45%**.

## ✔️ Key Features

- **Real-Time Availability Tracking:** Track empty parking spots in real-time across multiple zones with 95% accuracy. Slots are dynamically released as vehicles enter and exit.
- **Smart Discovery & Booking:** Search for parking based on your schedule and location. Book, cancel, and modify reservations seamlessly.
- **Automated Workflows:** Dynamic cost calculation, instant email confirmations, and digital PDF receipt generation.
- **Admin Dashboard:** A robust dashboard for administrators to monitor live bookings, analyze peak usage trends, and manage user feedback.
- **Security & Analytics:** Cross-checks complete vehicle history upon exit using registration numbers and alerts users if their vehicle is parked in a "No Parking Zone".
- **Find My Vehicle:** Easily track your parked vehicle's location inside a massive parking lot using its registration number and color.

## 🔑 Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** Python, Django
- **Database:** MySQL
- **Advanced Tech:** Machine Learning & Computer Vision for automated gate access and vehicle detection.
